# Audit Helper for Moderators

## Safety Declaration

- **Network**: No external network calls (except Mojang Auth/Session).
- **Telemetry**: Disabled. No analytics.
- **Filesystem**: Config file only (`config/betterdogs.json`).

## Commands

- `/betterdogs reload` (Planned): Reloads config.
- `/betterdogs debug` (WIP): Shows wolf genetics in chat.
